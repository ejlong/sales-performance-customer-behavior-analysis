This project uses the theLook eCommerce public dataset available in Google BigQuery.

Source:
https://console.cloud.google.com/marketplace/product/bigquery-public-data/thelook-ecommerce

Project ID: bigquery-public-data  
Dataset ID: thelook_ecommerce  

Description:
The theLook eCommerce dataset contains order, customer, product, and event data from a fictional eCommerce company.

The dataset is publicly available and maintained by Google Cloud.

The raw data is not included in this repository. All analysis was performed using SQL queries provided in the /sql folder.
